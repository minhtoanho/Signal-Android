# Features

Signal Android provides secure messaging with a comprehensive set of features.

## Core Messaging

### Text Messages

- End-to-end encrypted text messages
- Rich text formatting
- Message reactions
- Message quoting/replying
- Mentions in groups

### Media Messages

- Photos and videos (end-to-end encrypted)
- Voice messages
- Audio messages
- Documents
- Contact cards

### Message Features

- Disappearing messages (timer-based)
- Message editing
- Message deletion
- Read receipts
- Typing indicators

## Calls

### Voice Calls

- End-to-end encrypted voice calls
- High-quality audio (Opus codec)
- Speakerphone and Bluetooth support
- Call waiting

### Video Calls

- End-to-end encrypted video calls
- Group video calls
- Picture-in-picture mode
- Camera switching

### Call Features

- Call history
- Call notifications
- Call quality indicators
- Screen sharing (in development)

## Groups

### Group Types

| Type | Size | Features |
|------|------|----------|
| Small Group | Up to 1000 | Full encryption |
| Large Group (MMS) | Unlimited | Legacy support |

### Group Features

- Group admins
- Group avatars
- Group descriptions
- Invite links
- QR code invites
- Member permissions
- @mentions

- Identity key notifications
- Verified badges

### Privacy Settings

- Screen lock

- Read receipts toggle
- Typing indicators toggle
- Link previews toggle
